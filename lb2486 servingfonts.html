INTRODUCTION

Basically "Serving fonts" is nothing but the process of making different fonts available for download from google or internet. when we use the fonts in web pages,we can ensure that fonts are available and can be used texts from web pages.

Different methods of serving fonts

  • "Adobe fonts" and "Google fonts" are easy ways to use in serving fonts.
  • Font Face Observer is used to ensure that the texts in the webpages are easy to read while cumtom fonts are loading.
  • Fond Loading API, controls how fonts are applied in web pages.
  • Self-Hosting Fonts, In this method we can host our own server by this we can use our own custom fonts.
